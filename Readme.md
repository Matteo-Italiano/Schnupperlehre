3. Praktische Übung: Ein kleines Programm schreiben (30 Minuten)

Aufgabe 1: Begrüssungsprogramm

Beschreibung: Das Programm fragt nach dem Namen des Nutzers und begrüßt ihn.

Anleitung:

Erweiterung: Schnupperperson fügt eine Altersabfrage hinzu:

alter = int(input("Wie alt bist du? "))

else: print("Du bist noch nicht volljährig.")

Erweiterung 2: Schnupperperson soll das Programm also Loopen.

Aufgabe 2: Taschenrechner Einfach

print(f"Das Ergebnis ist: {zahl1 + zahl2}")

1. Einführung in HTML (15 Minuten)

Erklärung, was HTML ist:

"HTML (HyperText Markup Language) ist die Struktur einer Webseite. Es beschreibt, was auf der Seite angezeigt wird."

Grundlegende Tags:

<html>, <head>, <body>, <h1>-<h6>, <p>, <img>, <a>, <ul>/<ol>, <li>.

Beispiel:

<!DOCTYPE html> <html> <head> <title>Meine Visitenkarte</title> </head> <body> <h1>Max Mustermann</h1> <p>Willkommen auf meiner Seite!</p> </body> </html>

4. Praktische Übung: Erstellen einer persönlichen Visitenkarte (45 Minuten)

Aufgabe: Die Schnupperperson erstellt eine einfache Visitenkarte mit Name, Bild, Hobbys und einem interaktiven Button.

Schritt-für-Schritt-Anleitung:

HTML-Struktur erstellen:



<!DOCTYPE html>
<html>
<head>
    <title>Visitenkarte</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            text-align: center;
            background-color: #f4f4f4;
        }
        .card {
            border: 1px solid #ccc;
            padding: 20px;
            margin: 20px auto;
            width: 300px;
            background-color: white;
            box-shadow: 0 4px 8px rgba(0,0,0,0.1);
        }
        img {
            border-radius: 50%;
            width: 100px;
        }
        button {
            padding: 10px 15px;
            background-color: blue;
            color: white;
            border: none;
            border-radius: 5px;
            cursor: pointer;
        }
    </style>
</head>
<body>
    <div class="card">
        <img src="https://via.placeholder.com/100" alt="Profilbild">
        <h1>Max Mustermann</h1>
        <p>Hobbies: Programmieren, Kung Fu, Reisen</p>
        <button onclick="showMessage()">Mehr erfahren</button>
    </div>
    <script>
        function showMessage() {
            alert("Danke, dass du meine Visitenkarte anschaust!");
        }
    </script>
</body>
</html>

Schnupperperson passt den Code an:

Namen, Hobbys, Bilder Ändern

CSS ausprobieren → Schriftgrösse, Farben usw.

Javascript Funktion, welche den Namen alert() macht
